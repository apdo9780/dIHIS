About I-HIS

Project Overview

The Intelligent Hospital Information System (I-HIS) is a cutting-edge healthcare platform designed as part of the Professional Master's in AI in Healthcare (Term 2) course. It represents a comprehensive approach to integrating artificial intelligence with traditional hospital operations.

Objectives
  • ✓ Build a modern Flask-based healthcare information system
  • ✓ Implement Electronic Patient Records (EPR) management
  • ✓ Integrate AI-powered medical specialists progressively
  • ✓ Develop multi-agent collaborative AI systems
  • ✓ Demonstrate clinical decision support capabilities
Current Features (Week 1)
  • ✓ Patient Registration Module
  • ✓ Electronic Patient Records (EPR)
  • ✓ Hospital Receptionist AI
  • ✓ Patient Search & Management
  • ✓ Department Routing System
  • ✓ Appointment Scheduling
Technology Stack

Backend:

  • Python 3.x
  • Flask Framework
  • SQLAlchemy ORM
  • SQLite Database

Frontend:

  • HTML5 / CSS3
  • Bootstrap 5
  • JavaScript
  • Jinja2 Templates
Development Team

This project is developed as part of the Professional Master's Program in AI in Healthcare. Students work collaboratively to build an intelligent healthcare system that demonstrates practical applications of AI in medical settings.
